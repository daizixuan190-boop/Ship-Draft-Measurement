# A Lightweight Framework for Ship Draft Measurement via a Unified Detection Paradigm

This repository provides access information for the code, dataset, data split files, annotation files, and reproduction materials used in our study.

To support reproducibility and academic comparison, the complete project resources are available upon reasonable academic request. Researchers who wish to reproduce the results or conduct academic comparative studies may contact the authors through this repository and provide their name, affiliation, and intended research purpose.
